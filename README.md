# Amuse Web

This is a web-only version of the Amuse generative art application, ported from the Electron `spirograph-pro` project.

## Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run local server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## GitHub Setup

To push this to a new GitHub repository:

1. Create a new repository on GitHub (e.g., `amuse-web`).
2. Run the following commands:
   ```bash
   git remote add origin https://github.com/YOUR_USERNAME/amuse-web.git
   git branch -M main
   git push -u origin main
   ```
